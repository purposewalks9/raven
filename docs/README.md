# Nova Docs (content source)

Source content for the docs site (rendered by `website/`). Mirrors the
structure of `language/*.md` but rewritten for an external audience —
`language/` is the internal/compiler-team-facing spec, `docs/` is the
public-facing guide, tutorial, and API reference.

Suggested first pages once there's something to document:
- Getting started (install, `nova compile`, hello-world)
- Language tour (state, actions, UI nodes — using the login page as the example)
- Full-stack guide (server actions, routing)
- API reference (generated from `@nova/*` package types)
